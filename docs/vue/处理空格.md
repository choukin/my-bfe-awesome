# vue实现展示连续多空格
 
 vue中内容中间空格无论输入多少只会展示一个,可使用v-html来实现多空格展示

`&nbsp;` 英文空格 `&emsp;` 中文空格

```html
<div v-html="'价&emsp;&emsp;格'"></div>
```

或者使用white-space: pre; 直接使用空格
```html
<div style="white-space: pre;" v-html="'价      格'"></div>
```
　　

在uniapp中 <text>标签有space属性可展示连续空格

space 值说明

|值|说明|
|:-:|:-:|
|ensp|	中文字符空格一半大小|
|emsp	|中文字符空格大小|
|nbsp|	根据字体设置的空格大小|
